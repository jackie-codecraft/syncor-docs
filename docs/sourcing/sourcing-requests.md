# Sourcing Requests

## What Are Sourcing Requests?

Sourcing Requests let you request products that aren't in the standard catalog. Describe what you need, and the procurement team sources it for you.

### Lifecycle

| Status | Description |
|--------|-------------|
| **Submitted** | Request submitted for review |
| **In Review** | Procurement team is reviewing |
| **Sourcing** | Team is actively sourcing the product |
| **Completed** | Product sourced, request fulfilled |
| **Closed** | Request closed |

Click **Source** in the sidebar to access sourcing requests. This option is only visible if you have the sourcing permission.

---

## Creating a Sourcing Request

1. Navigate to **Source** in the sidebar.
2. Click **"+ Create New Request"**.
3. Fill in the form:

| Field | Required | Description |
|-------|----------|-------------|
| **Product Description** | Yes | Describe what you need in detail |
| **Estimated Quantity** | Yes | How many units you need |
| **Example Product Link** | No | URL to a similar product |
| **Upload Images** | No | Up to 10 reference images (drag-and-drop) |
| **Additional Comments** | No | Extra context or specifications |

4. Click **Create**.

:::tip
The more detail you provide, the faster the procurement team can find the right product.
:::

## Tracking Sourcing Requests

### List View

The main page shows: Request ID, Description, Quantity, Status, and Request Date. Click any row to view details.

### Detail Page

**Details Card** shows all submitted information: status badge, request ID and date, description, quantity, product link (opens in new tab), comments, and image gallery.

**Activity Card** is a real-time communication thread with the procurement team:

- Scrollable message history with sender name, type badge (User / System), timestamp, and content
- Images display as thumbnails; other files appear as clickable links
- Type and send messages in the input field
- Attach files via the paperclip icon (up to 5 files, max 20MB each)

:::tip
Press **Enter** to send. Use **Shift+Enter** for a new line.
:::
