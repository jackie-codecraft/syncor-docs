# Product Listings & Filtering

## Grid and List Views

Toggle between layout modes using the icons in the toolbar:

- **Grid View** (default) — Products as cards in a two-column grid
- **List View** — Products as horizontal rows

## Product Cards

Each product card displays:

- Product image, category badge, name, SKU
- Unit of measure and price (shows "N/A" if unavailable)
- Variant indicator (e.g., "9 Size") if variants exist
- Quantity selector (+/- buttons)
- **Add** button

## Sorting

Use the **Sort By** dropdown: **Newest** (default), **Name (A-Z)**, or **Name (Z-A)**.

## Pagination

The toolbar shows the current range (e.g., "Showing 1 - 10 of 10 Items"). Pagination controls appear at the bottom if there are more products.

## Sidebar Filters

When browsing a category or after a search, filter options appear in the left sidebar (brand, size, color, etc.). Use **Clear All** to remove all active filters.
