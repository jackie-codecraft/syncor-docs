# Search and Filter

Syncor's catalogue search helps you find the right product quickly, even when the catalogue contains hundreds or thousands of items.

## Basic Search

The **search bar** at the top of the catalogue panel accepts free-text queries. Type a product name, supplier name, part number, or keyword and press **Enter** (or click the search icon). Results are ranked by relevance.

**Tips for better search results:**

- Use specific terms — *"A4 copy paper 80gsm"* returns better results than *"paper"*
- Try the supplier's product code or SKU if you know it
- Partial matches are supported — *"laptop dock"* will match *"USB-C Laptop Docking Station"*

## Filtering Results

After searching (or while browsing a category), use the **Filter panel** on the left to narrow down results:

| Filter | Description |
|---|---|
| **Category** | Show only items from a specific product category |
| **Supplier** | Limit results to one or more preferred suppliers |
| **Price range** | Set a minimum and/or maximum unit price |
| **Lead time** | Show only items deliverable within your required timeframe |
| **In stock only** | Hide items that are currently out of stock |

Multiple filters can be applied simultaneously. Active filters are shown as tags at the top of the results — click the **×** on any tag to remove that filter.

## Sorting Results

Use the **Sort by** dropdown to order results by:

- **Relevance** (default for searches)
- **Price: Low to High**
- **Price: High to Low**
- **Supplier name (A–Z)**
- **Lead time: Shortest first**

## Saving a Search

If you regularly order the same items, you can save a search query as a **Favourite Search**:

1. Perform your search and apply any filters.
2. Click the **Save Search** button (bookmark icon) in the search bar.
3. Give the search a name (e.g. *"Monthly stationery order"*).
4. Access saved searches from the **Saved Searches** section in the catalogue sidebar.

## No Results?

If your search returns no results:

- Check your spelling and try a more general term
- Remove some filters to broaden the search
- Contact your Procurement Manager to request the item be added to the catalogue
